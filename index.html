☁️ クラウド連携 (GAS)

Google Apps ScriptのURLを入れてね。設定すると妻と自動でデータを共有できます。

↓このURLをLINE等で妻に送ると、妻は見るだけの画面を開けます。

📦 文字列でバックアップ

データを短い文字列にしてメモ帳などに保存・復元できます。

💡 GASのコード (コピペ用)

GASのエディタに貼り付けて、ウェブアプリとして「全員（匿名ユーザーを含む）」でデプロイしてね。

function doGet(e) {
  var d = PropertiesService.getScriptProperties().getProperty("data");
  return ContentService.createTextOutput(d || '{"debts":[]}').setMimeType(ContentService.MimeType.JSON);
}
function doPost(e) {
  PropertiesService.getScriptProperties().setProperty("data", e.postData.contents);
  return ContentService.createTextOutput('{"status":"ok"}').setMimeType(ContentService.MimeType.JSON);
}

📖 使い方マニュアル

1. 最初の設定（クラウド連携）

データを奥様と共有するために必要です。
① 「⚙️ 設定」タブの下の方にある「💡 GASのコード」を使って、ご自身のGoogleアカウントでGASのウェブアプリを作成します（アクセス権は「全員」）。
② 発行されたURLを、「☁️ クラウド連携 (GAS)」の入力欄に貼り付けます。
③ 「☁️ 雲へ保存する」ボタンを押せば設定完了です。

2. 奥様への共有

① 「⚙️ 設定」タブの「💌 妻に送る用URLをコピー」ボタンを押します。
② コピーされたURLをLINE等で奥様に送ります。
※ 奥様には「このURLを開いて、ブックマークかホーム画面に追加してね」と伝えてください。

3. 記録のつけ方（追加タブ）

① 「✍️ 追加」タブを開きます。
② 日付、内容、金額などを入力します。
※ 「総額計算から除外する」にチェックを入れると、一番上の「のこり総額」には含まれなくなります。
③ 「💖 記録を追加する」を押します。

4. 定期的な支払いの記録

携帯代やサブスク代など、毎月発生して終わりがないものの記録方法です。
① 「✍️ 追加」タブで内容を入力し、「📱 定期的なもの」にチェックを入れます。
② ホーム画面に項目が追加されるので、「✍️ お返しを記録」ボタンから毎月の支払額を入力していきます。

5. お返しをもらった時

① 「🏠 ホーム」タブで対象の項目を探し、「✍️ お返しを記録」ボタンを押します。
② 日付と金額を入力して記録します。
※ 残額が0円になると、自動的に「完済しました🎉」に変わります。

6. リストの整理

・各項目の「▲」「▼」ボタンで順番を自由に入れ替えられます。
・間違えた場合は「🗑️ 消す」ボタンで削除できます。
・上の「すべて」「返済中」などのボタンで見たい項目を絞り込めます。

7. 万が一のバックアップ

① 「⚙️ 設定」タブの「出力する (作る)」ボタンを押します。
② 出てきた文字列を「📋 コピー」し、スマホのメモ帳などに保存しておきます。
③ 機種変更後などに、その文字列を貼り付けて「入力する (復元)」を押せば元通りになります。
